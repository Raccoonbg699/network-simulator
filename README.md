# NetFlow Pro - Мрежов Симулатор (Дипломен Проект)

## Описание
NetFlow Pro е усъвършенствано JavaFX приложение за симулация на компютърни мрежи, проектирано като помощно средство за обучение по мрежови технологии. Позволява изграждане на комплексни топологии, детайлна конфигурация на устройства и интерактивна визуализация на алгоритми за маршрутизация.

## Основни Функционалности
- **Обширен каталог от устройства:**
  - *Мрежово оборудване:* Рутери (Cisco 1941, 2901, 2911, 1841), Суичове (2960, L3), Хъбове, Wireless АР и Firewall.
  - *Видеонаблюдение:* Пълна поддръжка на IP камери и NVR със симулиран видео фийд и управление на записи.
  - *Интелигентни системи (IoT):* Смарт дом (лампи, термостати, брави) и Индустриален IoT (PLC, сензори за налягане, клапани, конвейерни ленти).
  - *Крайни устройства:* PC, Laptop, Server, Printer, IP Phone и Смартфони.
- **Интуитивно изграждане на топология:**
  - Drag-and-drop интерфейс с автоматично центриране.
  - Инструменти за свободно рисуване (Pencil) и изтриване (Eraser) върху платното за бележки по топологията.
  - Пълна поддръжка на **Undo/Redo** (Ctrl+Z / Ctrl+Y) за графичните елементи.
- **Интелигентна Маршрутизация (Routing):**
  - Реализация на **Dijkstra алгоритъм** за изчисляване на най-кратки пътища.
  - **Визуализация на алгоритъма:** Режим "Стъпка по стъпка", който показва как Dijkstra обхожда мрежата в реално време (Highlight на текущи и посетени възли).
  - Пакетна симулация с визуална анимация по изчисления маршрут.
- **Интерактивна Операционна Система (Desktop UI):**
  - Компютрите разполагат с десктоп среда с вътрешна прозоречна система.
  - **Реален Уеб Браузър:** Вграден енджин (WebView) за сърфиране в реалния интернет.
  - **Имейл клиент:** Настройка на акаунти, изпращане и симулирано получаване на съобщения.
  - **Terminal & IP Config:** Пълна конфигурация на мрежовите параметри (IP, Mask, Gateway).
- **Персонализация и Управление:**
  - **Двуезичен интерфейс:** Пълна поддръжка на Български и Английски език.
  - **Теми:** Тъмна и Светла тема с автоматично превключване на всички компоненти.
  - **Persistence:** Автоматично запазване на прогреса при затваряне и зареждане при стартиране чрез JSON кеш.

## Технологичен Стек
- **Език:** Java 17+
- **Графична библиотека:** JavaFX 17 (Controls, FXML, Web, Media)
- **Зависимости:** Google Gson (за сериализация), JUnit 5 (за тестове)
- **Build System:** Maven

## Инсталация и Стартиране
1. Инсталирайте **Java 17** (или по-нова) и **Maven**.
2. Изтеглете проекта.
3. В основната директория изпълнете скрипта за стартиране:
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
   *Скриптът автоматично компилира проекта и го стартира в подходящ режим, заобикалящ евентуални модулни конфликти.*

## Файлова Структура
- `src/main/java/com/networksim/` - Основен сорс код.
- `src/main/resources/` - Езикови файлове (`.properties`), стилове (`.css`) и икони.
- `.network_cache.json` - Автоматично генериран файл със запазеното състояние на вашата мрежа.
